# Customer Churn Analysis

## 📊 Project Overview
This project analyzes customer churn behavior using a telecom dataset to identify key factors affecting customer retention.

## 🛠️ Tools Used
- Python (Pandas, NumPy, Scikit-learn)
- SQL

## 🔍 Key Insights
- Customers with month-to-month contracts have higher churn
- High monthly charges increase churn probability
- Short tenure customers are more likely to leave

## 📈 Model
Built a classification model with ~80% accuracy to predict churn.

## 📊 Dashboard
Includes Power BI dashboard showing churn trends and customer segmentation.
